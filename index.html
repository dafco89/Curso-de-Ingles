Test Final del Nivel A2

Este test evaluará tu dominio de todos los conceptos cubiertos en el nivel A2. Debes obtener al menos un 70% para aprobar y desbloquear el siguiente nivel.

Duración

60 minutos para completar todas las secciones del test.

Formato

50 preguntas divididas en comprensión auditiva, lectura, gramática y vocabulario.

Intentos

Tienes 3 intentos disponibles para aprobar el test.

Requisitos para realizar el test

Para poder realizar el test final del nivel A2, debes haber completado todos los módulos del nivel y sus respectivos tests.
